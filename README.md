# spring-gift-enhancement

## 1단계 구현사항
- 상품 카테고리 mvc 구현
- add.html 상품 추가시 카테고리 지정할 수 있도록
- 테스트코드 작성

## 2단계 구현사항
- 상품에 옵션 정보 구현, 상품과 옵션 모델 간 관계 고려(?)
- 옵션 네이밍 규칙
1) 공백 포함 최대 50자
2) 특수문자
    - 가능: ( ), [ ], +, -, &, /, _
    - 그 외 특수 문자 사용 불가
- 옵션 수량은 최소 1개 이상 1억 개 미만
- 옵션 이름 중복 허용 안함
- 컨트롤러 생성하여 HTTP 응답&요청 주고 받도록 구현
- 옵션 테스트 구현
